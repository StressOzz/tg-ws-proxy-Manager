# Установка

- в SSH выполните
```
sh <(wget -O - https://raw.githubusercontent.com/StressOzz/tg-ws-proxy-Manager/main/tg-ws-proxy-Manager.sh)
```
- В меню нажмите **1**

# Настройка Telegram

в **Telegram Desktop**:
- Настройки **→** Продвинутые настройки **→** Тип соеденения **→** Добавить прокси
- Выбрать **SOCKS5**
- В **Хост** указать **IP**, в **Порт** указать **порт**
- Сохранить

в **приложение Telegram**:
- Настройки **→** Данные и память **→** Настройки прокси **→** Добавить прокси
- Выбрать **SOCKS5**
- В **Сервер** указать **IP**, в **Порт** указать **порт**
- Галочка в верхнем правом углу

# Удаление
- в SSH выполните
```
sh <(wget -O - https://raw.githubusercontent.com/StressOzz/tg-ws-proxy-Manager/main/tg-ws-proxy-Manager.sh)
```
В меню нажмите **2**

При удалении, удаляются пакты **python3-light**, **python3-pip**, **git-http** и все зависимости связанные с этими пакетами
